# Data Directory

This directory is intended for storing the restaurant revenue dataset.

## Obtaining the Data

The dataset is automatically downloaded when you run the Jupyter notebook using the Kaggle API. 

### Manual Download

Alternatively, you can manually download the dataset:

1. Visit the [Restaurant Revenue Prediction Dataset](https://www.kaggle.com/datasets/mrsimple07/restaurants-revenue-prediction) on Kaggle
2. Download the dataset
3. Place `Restaurant_revenue.csv` in this directory

## Dataset Information

**Citation:**
MrSimple07. (2024). Restaurants Revenue Prediction [Data set]. Kaggle. https://doi.org/10.34740/KAGGLE/DSV/7420974

**Note:** This directory is gitignored to prevent large data files from being committed to version control.
